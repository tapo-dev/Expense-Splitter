# Expense Splitter

A prototype cross-platform mobile application designed to track shared expenses among roommates. The application allows users to log payments and calculates debts. Built using .NET MAUI.

## Technologies
- C#
- .NET MAUI
- SQLite

## How to Run
1. Clone the repository.
2. Open the solution file `ExpenseSplitter.sln` in Visual Studio 2022.
3. Ensure the .NET MAUI workload is installed.
4. Select an Android Emulator or a connected device.
5. Build and run the project.
